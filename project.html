Android Application(10/11 ~ 12/11)

Developed the android application based on the real salad bar.

- Allowed users to visually construct a salad from a visual representation of the salad bar. The users can then assemble their salad ingredients visually, add salad dressing and toss. Once done, the user can then order their salad from a restaurant that makes salads (e.g., Sweet Greens). Once the order is uploaded, the app can give the user an order number, post the order to the kitchen staff, inform the user when their order is ready, and create a check.
